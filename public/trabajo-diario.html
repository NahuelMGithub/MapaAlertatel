Como usar Trabajo diario

Esta pantalla sirve para empezar el dia con foco: primero ordenas la rutina, despues definis tareas concretas y reservás tiempo para avanzar el objetivo del mes.

  1. Ritual diario Marca los pasos base para revisar canales, mensajes y pendientes.
  2. Del ritual salen las tareas del dia Converti lo importante en tareas claras para hoy.
  3. Primero solo urgentes Resolvé lo que bloquea, vence o necesita respuesta inmediata.
  4. Trabaja 2 horas en OKR Reservá un bloque real para avanzar el objetivo mensual.
  5. Despues pendientes Con lo urgente y el OKR cubiertos, continua con el resto de tareas.
